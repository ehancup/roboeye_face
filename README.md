## Components u need
| Name | Amount |
|---|---|
| Arduino Uno | 1 |
| BreadBoard | 1 |
| Jumper Wire (Male to Male) | 10 |
| Oled Screen 128 x 64 | 1 |
| Push Button | 2 |
| Usb A to B cable | 1 |

## Package u need to install

| Name |
|---|
| [Adafruit 1306](https://github.com/adafruit/Adafruit_SSD1306) |
| [Adafruit GFX Library](https://github.com/adafruit/Adafruit-GFX-Library) |
| [RoboEyes](https://github.com/FluxGarage/RoboEyes) |
